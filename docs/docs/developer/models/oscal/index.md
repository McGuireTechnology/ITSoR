## Layer Orientation

OSCAL content is organized into three primary layers:

- **Control Definition**: defines reusable controls and catalogs.
- **Implementation**: captures how a system or offering implements selected controls.
- **Assessment**: documents how implemented controls are evaluated.

These layers align with the domain package structure under `backend/itsor/domain/models/oscal`.
