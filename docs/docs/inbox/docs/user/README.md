# User Guide

## Overview

Welcome to the ITSoR User Guide. This section explains how end users access and use the platform.

## Getting Access

1. Request access from your system administrator.
2. Confirm your account is active.
3. Sign in using your organization credentials.

## Core Tasks

- Find records
- View record details
- Update allowed fields
- Track status and history

## Troubleshooting

- If sign-in fails, verify your credentials and network/VPN.
- If data appears missing, refresh and confirm filters.
- If issues continue, contact support with screenshots and timestamps.

For technical triage and configuration checks:

- [Backend Auth Setup Guide](../backend-auth-setup.md)
- [Security Runbooks](../security-runbooks.md)
- [Frontend Auth Integration Guide](../frontend-auth-integration.md)
- [Backend Environment Variable Reference](../backend-env.md)

## FAQ

### Where do I report an issue?

Use your internal support channel or ticketing system.

### Who can edit records?

Edit permissions depend on your assigned role.
