# ITSoR
IT System of Record

ITSoR is a custom app-building platform focused on IT Management Information Systems.

## Platform Focus

ITSoR is designed to support core IT domains, including:

- CMDB (Configuration Management Database)
- ITAM (IT Asset Management)
- ITSM (IT Service Management)
- GRC (Governance, Risk, and Compliance)

## Data Exchange

ITSoR supports data exchange with external systems so information can be shared across tools and processes.

ITSoR apps work on desktop and mobile devices and support business use cases such as project management, operations, field work, human resources, sales, and marketing.
