# ITSoR
IT System of Record

ITSoR is a custom app-building platform focused on IT Management Information Systems.

## Platform Focus

ITSoR is designed to support requested IT domains, including:

- CMP (Cloud Management Platform)
- DCIM (Data Center Infrastructure Management)
- ESM (Enterprise Service Management)
- GRC (Governance, Risk, and Compliance)
- HAM (Hardware Asset Management)
- IDAM (Identity and Access Management)
- IGA (Identity Governance and Administration)
- ITAM (IT Asset Management)
- ITSM (IT Service Management)
- PAM (Privileged Access Management)
- SAM (Software Asset Management)
- UEM (Unified Endpoint Management)
- CMDB (Configuration Management Database)
- IPAM (IP Address Management)
- SIEM (Security Information and Event Management)
- SOAR (Security Orchestration, Automation, and Response)
- Vulnerability Management
- Observability (APM, NPM, DEM)
- Backup/DR
- CIMP
- Automation
- Orchestration

## Data Exchange

ITSoR supports data exchange with external systems so information can be shared across tools and processes.

## Features

### Capture and manage data

- Capture rich data with forms, barcodes, locations, signatures, and photos.
- Store data in ITSoR’s built-in database.
- Customize app features, UX, security, and branding.

### Automate operations

- Automate manual tasks so teams can focus on higher-value work.
- Send and automate email, SMS, and push notifications.
- Build workflow logic with triggers, conditions, and branches.
- Incorporate AI and machine learning, including intelligent document processing and optical character recognition.

### Integrate and exchange data

- Integrate with external systems through REST APIs.
- Exchange data across tools and processes.

### Govern with assurance

- Manage developers, users, teams, apps, and data across the organization.
- Select from a rich library of app governance policies or create your own.
- Set guardrails and gain visibility with a true no-code development platform.
- Apply built-in security controls, encryption, and verification with a Zero Trust approach.
- Retain full control of your data.

ITSoR apps work on desktop and mobile devices and support business use cases such as project management, operations, field work, human resources, sales, and marketing.
